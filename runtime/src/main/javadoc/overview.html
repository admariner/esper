Esper Runtime API - Runs EPL byte code

The runtime API consists of packages under com.espertech.esper.runtime.client. The classes and interfaces in the runtime API client packages are fully intended to be exposed to application code.

The classes under com.espertech.esper.runtime.internal are not intended for use by an application unless the application builds an extension. The internal API classes are not stable and can change significantly between versions.

Full documentation may be found at the home page.
