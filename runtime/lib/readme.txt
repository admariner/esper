Esper-Runtime Required Dependencies 
============================

slf4j-api-1.7.36.jar
- Simple Logging Facade for Java (SLF4J), for logging
- required for runtime
- see Apache v2 license in file esper_3rdparties_runtime.license

Optional runtime libraries
==========================

slf4j-reload4j-1.7.36.jar
- SLF4J implementation for Reload4j
- runtime, optional (consider using a different log provider)

reload4j-1.2.25.jar
- Reload4j Library for logging
- runtime, optional (consider using a different log provider)
